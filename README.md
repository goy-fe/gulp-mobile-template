## 集成功能

* SCSS
* Postcss
* Base64
* Babel
* Imagemin
* Clean-CSS
* Autoprefixer
* Uglify

支持配置定制，使用 `yaml` 来编写配置文件。

## 使用

> 需要 `Nodejs` 与 `Yarn` 环境。

### 开发环境

``` bash
$ yarn dev
```

### 打包项目

``` bash
$ yarn build
```
